## Autenticação

Todos os endpoints são protegidos por uma chave de API que deve ser fornecida no cabeçalho de cada requisição.  

<Note>
O cabeçalho HTTP esperado para a chave de API é x-api-key.
Falhas na autenticação (por chave ausente ou inválida) resultarão em um erro 401 Unauthorized.
</Note>

```json
"security": [
  {
    "x-api-key": ""
  }
]
```
